# Software Dev Management Game 🎮

Um jogo de gerenciamento onde você administra agentes que trabalham em sistemas de estabelecimentos, resolvendo bugs e concluindo melhorias.

## 🎯 Como Jogar

1. **Crie Agentes**: Use seu dinheiro para criar agentes com atributos aleatórios
2. **Selecione Atividades**: Escolha bugs ou melhorias para resolver
3. **Atribua Agentes**: Clique em um agente para trabalhar na atividade
4. **Ganhe Recompensas**: Complete atividades para ganhar dinheiro
5. **Desbloqueie Estabelecimentos**: Complete todos os sistemas para desbloquear novos estabelecimentos

## 🚀 Tecnologias

- HTML5 Canvas
- JavaScript (Vanilla)
- Canvas 2D API

## 🎨 Features

- Sistema de agentes com atributos customizáveis (Logic, Data Structure, Abstraction)
- Trabalho baseado em tempo real com barra de progresso
- Sistema de upgrade de agentes
- Múltiplos estabelecimentos e sistemas
- Interface com scroll independente
- Economia balanceada

## 🏃 Como Rodar

Basta abrir o `index.html` em um navegador moderno ou usar um servidor local:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server
```

Depois acesse: `http://localhost:8000`

## 📦 Deploy

Este projeto está pronto para deploy no Vercel, Netlify ou GitHub Pages.

## 🎮 Controles

- **Mouse**: Clique para interagir com botões e cards
- **Scroll**: Role o mouse para navegar pelas listas

## 💰 Economia

- Dinheiro inicial: $1000
- Custo de criar agente: $200
- Custo de upgrade: $100 por atributo
- Recompensas: Variam por atividade
